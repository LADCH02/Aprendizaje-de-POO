Preguntas de Reflexión

 

1. ¿Qué ventaja concreta te dio la herencia en este ejercicio? ¿Qué código
   evitaste repetir?
Bueno la ventaja principal para este codigo en particular es que existen varios tipos de ejercicios y cada uno ocupa ciertos datos diferentes pero al mismo tiempo
dichos ejercicios tienen una base en particular por eso mismo se acoplo la herencia un ejemplo es el cardio y las pesas, tienen la misma base los dos ocupan
un nombre y trabajan un grupo muscular, pero el rpe y aparte el peso es una particularidad de el ejercicio de pesas y del cardio no por eso mismo se hacen herencia para poder pasar metodos que se ocupan subclases
como este caso pero que al mismo tiempo dichas clases tienen sus propias particularidades.

   ¿Cuál versión se ejecuta?
